Write a program that accepts an ASCII art image and rotates it clockwise by a multiple of 90 degrees.

Input

The first line of input contains an integer specifying the number of degrees to rotate the image, R. R will always be a multiple of 90.

The remaining input is the ASCII-art image to be rotated. Any short lines should be padded with space (' ') characters to make the image rectangular.

Output

The output is the source image rotated clockwise by R degrees.

Example

STDIN:

90
o...o
.o.o.
..o..
..o..
..o..
..o..

STDOUT:

.....o
....o.
oooo..
....o.
.....o
